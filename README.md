# teamhermes.tk
## 수정하기
> 아래 명령어들은 node.js 설치 후
> cmd 또는 git bash 등을 사용해서 실행해 주세요.

```sh
npm install && cd teamhermes && npm install
```
명령어를 실행해서 dependencies를 설치합니다.
```sh
npm run serve
```
명령어를 실행하면 <http://localhost:8080>에서 실시간으로 결과물을 확인할 수 있습니다.

teamhermes/src/ 폴더에서 작업해주세요.
```sh
npm run deploy
```
